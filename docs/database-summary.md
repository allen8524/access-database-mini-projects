# Database Summary

## 1. address-book.accdb

주소록 관리용 Access 데이터베이스입니다.

예상 주요 구성:

| 구분 | 내용 |
| --- | --- |
| 주요 테이블 | 주소록, 가족 |
| 핵심 목적 | 연락처와 가족 정보를 관리 |
| 정리 내용 | 이름 데이터 랜덤화, 주요 테이블/필드명에서 `45` 접미사 제거 |

## 2. sales-management.accdb

판매관리 프로그램용 Access 데이터베이스입니다.

예상 주요 구성:

| 구분 | 내용 |
| --- | --- |
| 주요 테이블 | 종류, 제품, 판매, 판매내용 |
| 핵심 목적 | 제품 분류, 제품 정보, 판매 내역, 판매 상세 내역 관리 |
| 정리 내용 | 주요 테이블/필드명에서 `45` 접미사 제거 |

## Git 관리 주의사항

- `.accdb`는 바이너리 파일이므로 GitHub 웹 화면에서 내부 테이블 내용을 직접 확인하기 어렵습니다.
- 수정 이력을 깔끔하게 남기려면 Access에서 변경 후 파일 전체를 다시 커밋해야 합니다.
- 다른 사람이 내려받아 확인하려면 Microsoft Access 또는 Access Database Engine이 필요할 수 있습니다.
- 현재 파일은 Access 앱 없이 가능한 범위에서 정리된 버전입니다. 폼/보고서/숨김 객체 내부 문자열까지 완전 제거되었는지는 Access 앱에서 최종 확인하는 것이 안전합니다.
