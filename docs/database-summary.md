# Database Summary

## 1. 주소록 관리 데이터베이스

파일명: `databases/address-book.accdb`

주소록과 가족 정보를 관리하기 위한 Microsoft Access 데이터베이스입니다.

### 주요 테이블

| 테이블 | 설명 |
| --- | --- |
| 주소록 | 이름, 전화번호, 생일, 주소 등 연락처 기본 정보 관리 |
| 가족 | 주소록 데이터와 연결되는 가족 구성원 정보 관리 |

### 설계 목적

- 연락처 정보를 한 곳에서 관리
- 연락처별 가족 정보를 별도 테이블로 분리
- 주소록과 가족 정보를 관계형 구조로 연결

## 2. 판매관리 데이터베이스

파일명: `databases/sales-management.accdb`

제품 분류, 제품 정보, 판매 내역, 판매 상세 내역을 관리하기 위한 Microsoft Access 데이터베이스입니다.

### 주요 테이블

| 테이블 | 설명 |
| --- | --- |
| 종류 | 제품 분류 정보 관리 |
| 제품 | 제품명, 단가, 종류번호 관리 |
| 판매 | 판매일, 판매시간, 합계, 결제상태 관리 |
| 판매내용 | 판매번호별 제품, 수량, 금액 관리 |

### 설계 목적

- 제품 분류와 제품 정보를 분리 관리
- 판매 1건에 여러 판매 상세 내역을 연결
- 판매번호를 기준으로 판매 내역과 판매내용을 관계형으로 구성

## Git 관리 주의사항

- `.accdb` 파일은 바이너리 파일이므로 GitHub에서 코드처럼 변경 내용을 비교하기 어렵습니다.
- Access에서 수정한 뒤에는 파일 전체를 다시 커밋해야 합니다.
- 내부 확인은 Microsoft Access 설치 환경에서 진행하는 것이 가장 정확합니다.
